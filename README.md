# AlgoTutor
## 🌐 Live Demo
https://praveenhjr.github.io/AlgoTutor/

AlgoTutor is an AI-powered Data Structures and Algorithms tutor designed to help students learn through first-principles thinking.

It provides:
- Algorithm explanations
- Time & Space Complexity analysis
- Code examples
- Dry runs and step-by-step breakdowns
- Interactive AI-powered conversations

Built using HTML, CSS, JavaScript, and Google's Gemini API.
